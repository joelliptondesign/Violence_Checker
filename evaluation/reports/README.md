# Regression Comparisons and Engineering Reports

Current regression comparison is deterministic and allowed only when baseline and run share corpus, evaluation, and successor semantic schema families. It records proposition-addressed introduced and resolved differences, validation and policy changes, typed classifications, and aggregate counts. Markdown reporting derives only from repository artifacts and makes no provider request.

Creation-time comparisons and reports remain byte-immutable evidence under their original vocabulary. Cross-family comparison is explicitly incomparable; historical reports are never regenerated by current code.

The `true-north-live-*-001` artifacts preserve the first live evaluation. The `true-north-live-*-002` artifacts record the second live corpus evaluation, the run-to-run comparison, and the eight-fixture live evaluation. These are observed provider evidence and readiness assessments only; no baseline acceptance is implied.
